Employee verification service for online hiring and recruiting.

Inzazu is a bootstrapped startup located in Telaviv Israel and Draper Utah. Inzazu specializes in online background checks, drug screenings, employment and educational verfications for human resource departments and recruiting agencies.

The Problem

A rising trend in customer complaints reguarding clairity and basic navigation of the interface.

Project Overview

Re-factor and design a user friendly dashboard for the recruiter interface.

The current dashboard

Despite being a little dated and intimidating to understand, there are plenty of great features and tools accessible in this interface. Though you may need a little help finding them.

Hierarchy Improvements

Lets forget the details for now and focus on how to improve the navigation and hierarchy. I'd like to clarify the user flow and suggest a a few changes using the image carousel below.

Style Changes

Font: The downside to using the Gotham font is that each character is wider than most and when presented with a lot data like a dashboard space is a premium. I would suggest a font simialr to Roboto which is condensed, looks great and reads quickly.

Color: Using lighter whites and greys will lighten up the dashboard and compliment vibrant colors.

Logo: Inzazu had recenlty changed its name to Inzazu HR and had yet to include the HR into the logo, so I thought I might include a suggestion with a brighter green and darker blue.

Final Design

The changes stated feel minor however when put together the look and feel of the dashboard is completely new and hopefully more obvious to the customers user experience.

Conclusion

After presenting my redesign to board members they accepted my new dashboard layout as well as the more readable Roboto font and plan to move forward with the changes in the coming weeks.
